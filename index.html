Proyectos

AppDoggs

App de Razas de Perros: patrón MVVM, Consumo de datos vía API con 2 endpoints. Primer fragmento muestra un listado en RecyclerView. Segundo fragmento exhibe imágenes de la Raza Seleccionada en RecyclerView, con opción de favoritos. Implementación técnicas : Room para persistencia y Retrofit para networking.

PlantApp

App mpv de imagenes de Plantas: patrón MVVM, Consumo eficiente de datos vía API con 2 endpoints. Primer fragmento muestra un listado con imagenes en un RecyclerView. Segundo fragmento exhibe imágenes de la planta Seleccionada y mas detalles con la opcion de enviar un correo prediseñado. Implementación técnicas : Room para persistencia y Retrofit para networking.

ShoesTap

maqueta App tienda de zapatillas: patrón Mvp, app contiene un listado de imagenes preestablecidas. Primer fragmento muestra un listado de zapatillas con imagenes y precios en un RecyclerView. al selecionar nos lleva al Segundo fragmento, que exhibe la imágen y da mas detalles, la posiblidad de agregar al carrito e ir al carrito. el tercer fragmento es el carrito de compras, que muestra un listado de los productos agregados, los precios actualizados, ademas da la posiblidad de finalizar compra o vaciar el carrito.Implementación de técnicas : SharedPreferences.

AppMarte

App de terrenos de marte: patrón MVVM, Consumo de datos vía API Rest Primer fragmento muestra un listado de imagenes de terrenos en un RecyclerView. Segundo fragmento exhibe la imágen del terreno Seleccionado precio. Implementación técnicas : Room para persistencia, Retrofit y Gson.
